# <h1 align="center">**Industria Telecomunicaciones Republica Argentina**</h1>

### **Analisis 2020-2021**
El presente proyecto refiere a análisis estadísticos de la industria de telecomunicaciones de la Republica 
Argentina con datos suministrados por entes oficiales entre el año 2020 y 2021 
 
### Introduccion
Se realizo un relevamiento y analisis sobre datos de la industria de telecomunicaciones en Argentina, teniendo
como eje el servicio de Internet Fijo, Internet Movil con diversas estadisticas tales como cantidad de accesos,
velocidad promedio de acceso, penetracion de mercado entre otros 

Por otra se conto con datasets provistos por ente nacional de telecomunicaciones con diversa informacion de los 
servicios de Internet Fijo, Internet Movil

### Analisis exploratorio EDA
Se realizo analisis de los datasets para verificar la calidad de datos, se implementaron analisis de datos nulos,
datos con anomalias, distribucion de los datos entre otros analisis 

Se aplicaron las modificaciones necesarias para poder realizar un analisis eficiente de la informacion disponible 

### Dashborad
Se implemento dashboard con datos provistos en los datasets del ente nacional de telecomunicaciones, verificando 
diferentes metricas, tendencias del mercado

### KPI´s Industria
Se evaluaron 4 Kpi's sobre penetracion de mercado, accesos cada 100 hogares, accesos segun tecnologia, evaluacion de
velocidad de servicio, determinando metricas y umbrales de puntos críticos

### Conclusiones
UMBRALES
Se establecen umbrales para diferentes metricas , de manera de tener un control de puntos críticos que pueden estar 
excluidos en analisis paralelos como KPI comparativos entre trimestres 
Los umbrales son una estrategia para detectar rapidamente puntos críticos en cuanto a objetivos generales de la empresa

DISPARADORES
Se establecen análisis paralelos sujetos a los umbrales para poder indentificar posibles causas o circunstancias de 
los puntos crítícos detectados entre ellos posibles problemas del servicio de acceso a internet en cuanto a su performance
